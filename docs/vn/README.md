# Tài liệu Hướng dẫn (VN)

Chào mừng bạn đến với hệ thống tài liệu hướng dẫn sử dụng và phát triển. Đây là trình biên tập bài giảng tương tác hiện đại, hỗ trợ kéo thả và kiến trúc plugin linh hoạt.

## 📚 Danh mục Tài liệu

Để tìm hiểu chi tiết về từng khía cạnh của dự án, vui lòng chọn một trong các mục dưới đây:

### ⚙️ [Kiến trúc & Kỹ thuật](ARCHITECTURE.md)
Tìm hiểu về cách hệ thống vận hành, luồng dữ liệu (Zustand), và cấu trúc thư mục của dự án.

### ✨ [Tính năng & Công cụ](FEATURES.md)
Khám phá toàn bộ các tính năng từ soạn thảo chữ trực tiếp, chèn âm thanh, video, đến việc tạo các câu hỏi Quiz tương tác.

### 🎨 [Giao diện & Trải nghiệm (UI/UX)](UI_UX.md)
Tìm hiểu về triết lý thiết kế, hệ thống màu sắc, hoạt ảnh và cách tối ưu hóa trải nghiệm người dùng trên Canvas.

### 🧩 [Danh mục Thành phần (Elements)](ELEMENTS.md)
Tra cứu chi tiết về danh sách các vật thể (văn bản, hình khối, media) và các thuộc tính đi kèm của chúng.

---

## 🚀 Bắt đầu Nhanh

Nếu bạn là nhà phát triển mới tham gia dự án, hãy đảm bảo bạn đã đọc file [**RULE.md**](RULE.md) ở thư mục này để nắm rõ các quy tắc viết code (như nguyên tắc DRY và giới hạn số dòng file).

### Cài đặt
```bash
npm install
npm run dev
```

### Triển khai Sản phẩm
```bash
npm run build
```
